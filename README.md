# Terac Autonomous Research Platform

Terac is an AI-native autonomous research platform built on Google Cloud that automates participant sourcing, interviewing, transcription, analysis, and insight generation for product and user research teams.

## How Terac Uses Google Cloud

- **Cloud Run**: Runs scalable microservices for voice, video and text interview processing.
- **Cloud Storage**: Stores research media files, transcripts and structured outputs.
- **Firestore**: Maintains participant metadata, scheduling, and real-time session data.
- **BigQuery**: Aggregates transcripts, behavioral signals and insights into fast, queryable datasets for analysis.
- **Identity Platform**: Provides secure authentication, multi-tenant access control, and user management.

## Platform Overview

Terac combines automated interview orchestration, AI-driven insight extraction, and a unified research data pipeline powered entirely by Google Cloud. Product teams use Terac to run user studies at scale, generate insights quickly and maintain full compliance with security and privacy standards. This page provides a public overview required for Google Cloud Solution Qualification.
